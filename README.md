\# DocBot — AI Document Assistant



Upload any document and ask questions about it. Powered by Google Gemini embeddings, Claude AI, and Pinecone vector search.



\## What it does

\- Upload PDF, DOCX, TXT, JPG, or PNG files

\- Automatically generates a document summary and suggested questions

\- Ask natural language questions and get accurate answers from the document

\- Supports scanned documents via Google Cloud Vision OCR



\## Tech Stack

\- \*\*Backend:\*\* Python, Flask

\- \*\*AI:\*\* Claude (Anthropic) for answers, Gemini for embeddings

\- \*\*Vector DB:\*\* Pinecone

\- \*\*OCR:\*\* Google Cloud Vision

\- \*\*Frontend:\*\* HTML, CSS, JavaScript
